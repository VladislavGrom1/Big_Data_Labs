# Системы обработки больших данных
## Лабораторная работа № 1
### Разведочный анализ данных с помощью PySpark
#### Цель и задачи работы:
1. Познакомиться с понятием «большие данные» и способами их обработки;
2. Познакомиться с инструментом `Apache Spark` и возможностями, которые он предоставляет для обработки больших данных.
3. Получить навыки выполнения разведочного анализа данных использованием `pyspark`.

## Лабораторная работа № 2
### Машинное обучение на больших данных с использованием фреймворка Apache Spark и библиотеки SparkML
### Вариант №3.
### Модель регрессии: GPTRegressor
### Модель бинарной классификации: LogisticRegression
#### Цель и задачи работы: 
1. Обучить и провалидировать распространённые модели машинного обучения с использованием датасета большого объёма.
2. Познакомиться с особенностями обучения моделей ML на данных большого объема.
3. Получить представление о реализации основных этапов машинного обучения в библиотеке `SparkML`.
4. Познакомиться с организацией подготовки признаков, обучения и валидации модели как воспроизводимого конвейера.
5. Получить навыки создания программного кода на `pyspark` для проведения основных этапов машинного обучения.
6. Научиться анализировать качество модели, обученной на больших данных.
